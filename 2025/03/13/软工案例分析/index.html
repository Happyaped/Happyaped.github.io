[I.2]个人作业 软件案例分析

我选择QQ音乐作为调研目标

第一部分 软件调研评测

  • 使用QQ音乐主要有三种方式，一种是已知目标音乐在搜索框中直接输入进行搜索，第二种是目标音乐在播放但是不知道名字，可以使用听歌识曲的功能，第三种是可以选择根据风格进行推荐其他音乐的方式，基本上解决了客户需要找到目标歌曲以及需要推荐新音乐的需求
  • 从数据量而言，比起网易云音乐和Apple Music是有很大缺陷的，比如有很多b站上的自创歌曲以及外国的歌曲QQ音乐缺乏版权，只是能够覆盖主流歌手的歌曲
  • 从界面而言，QQ音乐总体上算是简洁，主要功能都标注的非常清楚，但是有些功能还是延续了QQ的花里胡哨，比如说底下的直播功能就有些莫名其妙
  • 从功能而言，包含了比较重要的听歌识曲功能还有根据自己的歌单进行推荐的功能，还是非常方便的，缺点如上文所言，有些莫名其妙的功能比如说直播
  • 从准确度而言，如果是有版权的歌曲的话，搜索非常准确
  • 从用户体验来说，我感觉很棒，简洁的界面加上推荐歌曲功能已经满足了我对于音乐软件的需求
  • 我希望QQ音乐可以有更多的版权然后追求界面的极简，删去不必要的功能
  • 我希望推荐系统能够更加准确，因为有些时候就是推荐的流行歌而不是根据歌单风格推荐
  • 采访对象是wdq老师软工班上的丁广晴同学，他的需求是听歌和根据歌单推荐歌曲还有歌曲导入
  • 我的结论是好，还不错
  • 原因如下
  • 基本满足了用户需求，页面相对简洁
  • 有少量花哨的界面，版权有些不足

Bug分析与检测

  • 手机版QQ音乐版本 14.2.0.8
  • 手机型号HONOR 60
  • 操作系统MagicOS 8.0(基于Android 14)
  • Bug1
  • 手机端的阅读听歌好友排名显示错误，需要重新登录才会正确显示
  • 可复现性和步骤：
  • 首先需要点击听歌报告进行授权
  • 授权后发现还是排名999
  • 退出后重新登录发现正确排名
  • 猜测原因：可能听歌的信息和排名在登陆的时候进行获取从而导致授权后不重新登录就无法获取，我觉得属于是测试把关不严
  • 严重性：一星
  • 从用户体验角度有些影响，从安全性和系统功能来说没有问题
  • Bug改进意见：
  • 可能实时刷新可以解决问题
  • Bug2
  • 电脑端无法显示歌曲的实时弹幕
  • 可复现性和步骤
  • 因为手机端可以显示歌曲的实时弹幕，但是电脑端无法查看
  • 猜测原因：可能是现有的UI设计不支持展示实时弹幕，我认为属于具体的设计质量不高
  • 严重性：二星
  • 从用户体验角度有较大影响，因为热衷于某些歌曲的时候希望在听歌的时候引起大家的共鸣可以看实时弹幕
  • Bug改进意见：设计一个新的UI界面支持实时弹幕显示

分析

假设一个团队已经对技术非常熟悉而且每天工作8h

功能 天数 注释
搜索播放收藏评论音乐 40天 属于基础功能
歌单展示 10天 包含了各种排序，也是基础功能之一
歌曲推荐 发现歌单 80天 设计算法推荐等知识，属于高级功能
账号管理和设置 15天 基础必要功能
社区功能 40天 包含了很多细小的板块，需要精心测试
听歌识曲 15天 感觉这种功能有相关的轮子
  • 优势
  • 使用QQ登录和微信登录，不用额外注册账号
  • 可以看到QQ和微信好友的歌单进行分享和交流
  • 接入了DeepSeek接口，提供了AI作曲功能给普通用户创作音乐机会和门槛降低
  • 劣势
  • UI设计比较繁杂，某些功能莫名其妙
  • 评论和社区缺乏管理，某些社区有饭圈文化和捧踩强烈，有些戾气
  • 音乐版权少
  • 综合排名，我认为是第二或者第三的位置，功能还算齐全好用，但是年费会员的价格不如Apple Music的家庭套餐香
  • 提高的建议：
  • 删去莫名其妙的功能，比如下面的直播功能，降低年费会员的价格，多买些版权

建议和规划

  • 2020年10月，qq音乐的月活跃用户数量多达20122万（2亿多）人，在线服务付费用户约7000万人
  • 网易云音乐，Apple Music，酷狗音乐，Spotify
  • 酷狗音乐：老牌的音乐软件，没有太多突出的地方
  • QQ音乐：引入了QQ和微信登录，也有社区功能，版权比剩下两个多，有腾讯当后背，但是不如网易云新潮和社交属性
  • 网易云：社区功能强大，很多B站上自创的歌也能听，也有很多小的创作者，目标对象是年轻人
  • 针对于核心用户我认为QQ音乐适合于00后用户，主要是从小学生到研究生，爱好流行歌曲，无稳定收入，潜在需求是听歌放松学业压力
  • 针对于用户群体，QQ音乐用户之前的关系在于音乐社区，大家会对于有相同爱好的音乐进行讨论分享，也有特定音乐文化的社区交流的意愿更强，比如ACG音乐社区 QQ音乐会员和B站大会员购买有联动，而且不同音乐软件之间可以实现歌单的互相导入
  • 新功能：
类别 描述
N 让音乐社区的氛围缺少戾气，管理减少饭圈文化
A 推出音乐社区管理员机制，类似于贴吧吧主
B 可以让用户选择管理员来管理社区氛围
C 更好的社区氛围可以让更多相关音乐爱好者来选择这款听歌软件
D 借助腾讯的各个产品来宣传QQ音乐社区的管理员机制
类别 描述
N 在QQ音乐中听到B站的创作歌曲
A 和B站合作，可以单独实现B站热曲功能界面
B 在QQ音乐中听到更多自创歌和热门歌
C 不只有大歌手的流行歌曲，还有up主的粉丝来引流
D 借助腾讯和B站一起来宣传
